# Quiz Maker - پلتفرم آزمون آنلاین

## معرفی پروژه
سیستم مدیریت آزمون پیشرفته با قابلیت‌های:
- مدیریت کاربران
- ساخت و مدیریت آزمون
- امتیازدهی هوشمند
- گزارشگیری

## پیش‌نیازها
- PHP 7.4+
- Composer
- MySQL 5.7+
- Extension‌های PHP:
  * PDO
  * JSON
  * MBString
